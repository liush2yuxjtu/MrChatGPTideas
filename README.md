# MrChatGPTideas

一个轻量级仓库，用于收集、整理和迭代与 ChatGPT 相关的想法、提示词、工作流和实验。

## 项目目的

这个仓库可以作为个人 AI 想法实验室，用来记录和沉淀：

- ChatGPT 提示词想法
- AI 助手工作流概念
- 自动化实验
- 产品或原型笔记
- 可复用的人机交互模式
- 未来项目的灵感种子

## 仓库结构

仓库结构可以随着想法演进持续调整。建议的组织方式如下：

```text
.
├── README.md
├── ideas/
│   ├── prompts.md
│   ├── workflows.md
│   └── products.md
└── experiments/
    └── notes.md
```

## 建议的想法记录格式

每个想法可以使用下面的模板进行记录：

```markdown
## 想法标题

**问题：**  
这个想法要解决什么问题？

**概念：**  
核心思路是什么？

**使用场景：**  
谁会在什么情况下使用它？

**下一步：**  
最应该先验证或构建什么？
```
